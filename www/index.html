Perjalanan Karier: Dari O hingga menjadi Junior Penetration Tester

Sepanjang perjalanan karier seseorang, pendidikan memainkan peran yang sangat penting dalam membentuk masa depannya. Ini adalah kisah inspiratif tentang perjalanan seorang individu yang dimulai dari nol dan yang kini menjadi Junior Penetration Tester.

Bootcamp

Berbekal ilmu pengetahuan mengenai jaringan di Sekolah Menengah Kejuruan (SMK), saya memutuskan untuk mengikuti Bootcamp Cyber Security. Di sini, saya mempelajari tentang Cyber Security dan mendapatkan sertifikasi. Saya juga mempelajari tentang HTML, CSS, JavaScript, PHP, dan Laravel. Saya juga mempelajari tentang tools yang digunakan dalam dunia Cyber Security seperti Burpsuite, Kali Linux, dan Metasploit. Saya juga mempelajari tentang pengembangan aplikasi web dan System Administration menggunakan Linux.

Bug Bounty Experience

Pada saat itu saya mencoba mengimplementasikan hasil belajar saya dengan melakukan Bug Bounty. Lalu saya menemukan sebuah bug pada sebuah website yang berbasis PHP. Saya melaporkan bug tersebut dan mendapatkan reward sebesar $100. Dari sini saya semakin termotivasi untuk terus belajar dan mengembangkan diri saya.
